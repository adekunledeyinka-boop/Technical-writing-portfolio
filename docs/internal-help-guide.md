Internal Cloud Operations Dashboard - User Guide
## Overview
This guide explains how the transaction managers, partner teams use the one pipeline application dashboard to add new pipeline sites, delete, manage and update existing pipeline sites
## Audience
- Real Estate Managers
- Partner Team Members
## Prerequisites
- AWS account access
- L3-L4 Members reporting to Jack bill and India Joan
  ## Navigate the Dashboard
  1.Log in to the dashboard using your AWS SSO
  2. From top: redirect back to the homepage, access other applications for Regions 1, 2 and 3
  3. From bottom: Interactive tabs takes you directly to more detailed help guide on tools, slack channel, service ticket to log bugs
  3. From Left: Tools to input pipeline site, 
  4. From Right: Layer list for all pipeline sites in order of priority
  ## Troubleshooting
  ** Issue** Unable to Login to dashboard
  ** Solution** Verify you belong to the right accrs group and report to Jack bill and India Joan
